# GitHub Profile Analyzer

A React application that allows users to search GitHub profiles and view profile information along with repositories.

## Features

* Search any GitHub user
* View profile details
* View followers and following count
* View public repositories
* View repository stars and language
* Direct links to GitHub profile and repositories

## Tech Stack

* React
* Vite
* Axios
* GitHub REST API

## Installation

```bash
git clone https://github.com/abhigyan-oss/github-profile-analyzer.git

cd github-profile-analyzer

npm install

npm run dev
```
